注释【重点】
    概念和作用：对程序/代码解释说明的一段文字，给开发人员看的，帮助开发人员理解程序/代码。
    分类：
        单行注释：
            语法：//注释内容
            应用场景：解释说明某一句代码。
            快捷键：ctrl+/
        多行注释：
            语法：/*多行注释内容*/
            应用场景：解释说明某一段代码。
            快捷键：ctrl+shift+/
        文档注释：
            语法：/**多行注释内容*/
            应用场景：解释说明某个类或者方法等

字面量
    概念：数据在程序中的书写格式。
    分类：
        整数              666,-88
        小数              13.14,-5.21
        字符              用单引号引起来，''中的字符有且仅有一个。例如：'我'、'a'、'1'
        字符串            用双引号引起来，""中的字符可以有0个或者多个。例如："我爱西安"、""、"a"
        布尔值            true表示真，false表示假
        空值              null
        特殊字符字面量     \t  \n

变量【重点】
    概念和作用：内存中的一块区域，用来保存程序中经常发生变化的数据。
    特点：变量中保存的数据可以发生变化。
    语法：数据类型 变量名 = 数据(值)。例如：int age = 20;  double money = 9.9;
    重新赋值：变量名 = 新数据(值)。  //注意：重新赋值的时候在变量名前面不能写数据类型。

数据类型
    作用：规定变量能存储什么类型的数据
    分类：【重点】
        基本数据类型：4类8种
            整型
                byte    1个字节    字节型，取值范围-128~127
                short   2个字节    短整型
                int     4个字节    整型，整型的默认类型
                long    8个字节    长整型
            浮点型
                float   4个字节    单精度浮点型
                double  8个字节    双精度浮点型，浮点型的默认类型
            字符型
                char    2个字节    字符型
            布尔型
                boolean 1个字节    布尔型，布尔型只有两个值：true和false
        引用数据类型：数组、对象等。String字符串属于引用类型。
            String name = "张益达";
    类型转换
        自动类型转换：类型范围小的变量/数据值 可以 直接赋值给类型范围大的变量存储。【重点】
            byte-->short-->int-->long-->float-->double
                    char-->int-->long-->float-->double
            表达式中的自动类型转换
                byte、short、char-->int-->long-->float-->double
                注意1：在表达式中，byte、short、char是直接转换成int类型参与运算的。
                注意2：表达式的最终结果类型由表达式中的最高类型决定。
            快捷键：  表达式.var+回车，  生成表达式等号左边的变量。例如  x+y.var+回车--->生成int z =x + y
        强制类型转换：类型范围大的变量/值 不能 直接赋值给类型范围小的变量报错，需要强制类型转。
            语法：小范围类型 变量 = (小范围类型) 大类型范围的变量/值;
            注意1：强制类型转换可能造成数据(丢失)溢出；
            注意2：浮点型强转成整型，直接丢掉小数部分，保留整数部分返回

关键字和标识符【重点】
    关键字：Java自身用到的一些具有特殊含义的英语单词。例如：public、class、int、long、double...
    标识符：给包、类、变量等取的名字
        规则：
            1、一般由数字、字母、下划线(_)和美元符($)等组成
            2、不能以数字开头，不能是关键字，不能包含一些特殊字符(&、%、#...)
            3、变量名：建议首字母小写，后面每个单词首字母大写，满足"小驼峰模式"
                小驼峰模式：当名称由多个单词构成时，第一个单词首字母小写，后面每个单词首字母大写。
            4、类名：建议所有单词首字母都大写，满足"大驼峰模式"
                大驼峰模式：：当名称由多个单词构成时，每个单词首字母都大写。
debug技术【重点】
    debug:通过控制代码一行一行执行，定位和分析代码中存在的错误的一种手段。
    使用步骤：
        第一步：在代码的左侧行号处点一下形成断点。注意不要设置在注释行上。
        第二步：以debug方式运行程序，程序会停在断点行。
        第三步：控制程序一步一步执行，观察变量的变化情况以及输出的结果。

键盘录入技术【近期重点】
    Scanner类：Java给我们提供的扫描器类，需要new这个类的对象（根据图纸生产扫描器对象）才可以使用。
    使用步骤：
        第一步：导包，诉程序去JDk的哪个包中找Scanner扫描器。idea可以自动导包。
        第二步：创建扫描器Scanner对象，会自动导包。
            Scanner sc = new Scanner(System.in);
        第三步：用方法等待用户输入不同类型的数据。提示并获取用户的内容。
            System.out.println("请输入姓名：")
            String name = sc.next();
            System.out.println("请输入年龄：")
            int age = sc.nextInt();
