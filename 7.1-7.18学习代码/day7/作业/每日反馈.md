①、变量的命名规则和规范是什么？
  规则：命名只能使用字母、数字、下划线和美元符号（$）。
  命名不能以数字开头。不能使用关键字
  命名对大小写敏感，区分大小写
  规范：变量名应具有描述性，能够清晰表达变量的含义。
  使用驼峰命名法
②、基本数据类型分为哪几种?
  number数字型 string字符串型 boolean布尔型 null空类型 undefined未定义类型
③、模板字符串有什么作用？
  在字符串中拼接变量
④、运算符有哪几种？
  算数运算符+、-、*、/、%
  赋值运算符==、===、!=、!==、>、<、>=、<=
  自增自减运算符++、--
  逻辑运算符&&、||、!
  三元运算符 条件表达式? 值1:值2
⑤、请写出if三种分支结构的语法?
  单分支
  if (条件) {
    // 条件为真时执行的代码
  }
  双分支
  if (条件) {
    // 条件为真时执行的代码
  }else {
    // 条件为假时执行的代码
  }
  多分支
  if (条件) {
    // 条件为真时执行的代码
  }else if (条件) {
    // 条件为真时执行的代码
  }else {
    // 条件都不满足时执行的代码
  }
